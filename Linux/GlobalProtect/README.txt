Install commands:
  sudo apt-get install libqt5webkit5
  sudo dpkg -i GlobalProtect_UI_deb-5.2.1.0-7.deb

  When prompted for a portal address, enter portal address.
  
  Once installation is complete, GlobalProtect will appear in 
  your menu bar at the top of your Linux system. If it does not
  appear, you can get the interface to appear by entering
  [globalprotect launch-ui]
  
  protal: vpn-cph.unity3d.com
